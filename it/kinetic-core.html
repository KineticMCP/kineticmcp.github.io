Perché Kinetic Core?

Auth Gestita

Supporto nativo per JWT Bearer Flow (Prod) e Username/Password (Dev). Gestione automatica del rinnovo token.

Bulk API v2

Carica o scarica milioni di record. Gestione automatica di batch, polling e retry backoff.

Metadata as Code

Crea Campi e Oggetti Custom programmaticamente. Definisci il tuo schema dati direttamente in Python.

Semplice, ma potente.

Abbiamo progettato Kinetic Core per essere intuitivo per chi inizia, ma robusto per chi scala. Type-hinting completo, logging contestuale e gestione errori enterprise.

  • Type-safe e Autocomplete friendly via Pylance
  • Zero dipendenze pesanti (solo `requests`, `pyjwt`, `cryptography`)
  • Supporto Docker e Serverless
main.py 
from kinetic_core import JWTAuthenticator, SalesforceClient

# 1. Autenticazione Enterprise
auth = JWTAuthenticator.from_env()
session = auth.authenticate()

# 2. Inizializza Client
client = SalesforceClient(session)

# 3. Query Semantica
accounts = client.query(
    "SELECT Id, Name FROM Account WHERE Industry = 'AI'"
)

# 4. Creazione Record
new_ops = client.create(
    "Opportunity", 
    {
        "Name": "Kinetic Pilot",
        "StageName": "Prospecting",
        "CloseDate": "2025-12-31"
    }
)

print(f"Successo! ID: {new_ops}")
