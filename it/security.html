  • No Training: Inputs e Outputs non sono mai usati per allenare i modelli Claude.
  • Data Retention: Opzioni "Zero-day retention" disponibili per alta compliance.
  • Isolamento: Gli utenti Enterprise operano in ambienti dedicati e isolati.

L'Architettura "Zero-Trust"

Abbiamo progettato KineticMCP per essere invisibile ai tuoi dati.

Self-Hosted Docker

Non invii dati ai nostri server. Esegui il container nel tuo VPC (AWS, Azure, On-Prem). Le tue chiavi, la tua infrastruttura.

Zero Data Retention

KineticMCP è stateless. Riceve una query, la traduce in chiamate Salesforce API, restituisce il risultato e dimentica subito tutto.

Permessi Utente

L'Agente si autentica come uno specifico Utente Salesforce (es. "Integration User"). Non può fisicamente accedere a campi bloccati da Profili o Permission Sets.

Audit Trail Completo

Ogni singola azione compiuta dall'AI viene loggata nella tua console locale (stdout/stderr). Invia i log a Datadog, Splunk o CloudWatch per la compliance.

Air-Gap Ready

Serve sicurezza estrema? Esegui KineticMCP senza accesso a internet, collegandolo a LLM locali (Llama 3 via Ollama) invece che ad API pubbliche.

Open Source Core

Non ti fidare delle scatole nere. Ispeziona la logica del nostro adapter su GitHub. Verifica esattamente come interpretiamo i Metadata e le query SOQL.
