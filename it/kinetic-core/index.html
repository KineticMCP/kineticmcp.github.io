Il Codice è la Verità.

Dimentica i Change Sets e le interfacce point-and-click. Con Kinetic Core, la struttura del tuo CRM vive nel repository Git. 

# Definizione Schema in Python
from kinetic_core.metadata import CustomField

# Creo il campo "Livello AI"
ai_score = CustomField(
    sobject="Account",
    name="AI_Score__c",
    type="Number",
    precision=18,
    scale=2,
    description="Punteggio generato dal modello predittivo"
)

# Deploy in Org
client.metadata.deploy(ai_score)
